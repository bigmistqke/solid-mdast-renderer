# solid-lezer-markdown

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

lezer based markdown renderer

> **Note** After using this template, you have to search and replace all `solid-lezer-markdown` and similar strings
> with appropriate texts.
>
> `solid-lezer-markdown` should be a **kebab-case** string representing the name of you monorepo.
>
> `lezer based markdown renderer` should be a **Normal case** string with the description of the repository.
>
> `bigmistqke` should be a **kebab-case** string from your profile URL.

## Quick start

Install it:

```bash
npm i solid-lezer-markdown
# or
yarn add solid-lezer-markdown
# or
pnpm add solid-lezer-markdown
```

Use it:

```tsx
import solid-lezer-markdown from 'solid-lezer-markdown'
```
